# 综治中心矛盾纠纷管理系统

## 项目概述

基于微服务架构的一站式矛盾纠纷闭环管理平台，支持多端接入、全流程处理、智能辅助决策。

## 技术栈

### 前端
- **管理端**: Ant Design Pro + React 18 + TypeScript + ECharts
- **自助终端**: Vue 3 + Element Plus + Vite (触屏适配)
- **小程序**: 腾讯Kuikly (Kotlin Multiplatform)

### 后端
- **微服务框架**: CloudWeGo (Hertz + Kitex + Netpoll)
- **工作流引擎**: Flowable
- **消息队列**: Apache RocketMQ

### 数据存储
- **关系型数据库**: TiDB (分布式)
- **缓存**: Redis Cluster
- **搜索引擎**: Elasticsearch
- **对象存储**: MinIO
- **向量数据库**: Milvus

### AI能力
- **大语言模型**: DeepSeek
- **向量检索**: Milvus

## 核心功能模块

### 1. 自助登记终端 (Vue3)
- 身份证读卡器自动读取信息
- 三级纠纷类型选择
- 证据拍照上传
- 回执二维码打印
- 离线模式支持

### 2. 小程序端 (Kuikly)
- 纠纷在线登记
- 进度实时查询
- 满意度评价
- 在线补充材料
- 一键催办
- 智能法律咨询 (DeepSeek)

### 3. 管理端 (React)
- 案件智能分派
- 调解过程记录
- 多级审批 (调解员→组长→主任)
- 审批进度时间轴
- 自动催办与超时升级
- 视频调解
- 电子签章
- 热力图展示
- 绩效考核
- 数据大屏
